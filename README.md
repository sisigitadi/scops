# SOCOps Dashboard - Tactical Operations Interface

Sistem pemantauan operasional SOC (Security Operations Center) yang dirancang untuk memberikan visibilitas real-time, manajemen insiden, dan koordinasi tim dalam lingkungan investigasi digital.

## 🚀 Fitur Utama

- **Real-Time Dashboards**: Visualisasi performa operasional dan metrik keamanan secara langsung.
- **Forensic Alert Grid**: Manajemen alert dengan tingkat kepadatan data tinggi untuk efisiensi investigasi.
- **Tactical Operations Hub**: Pemantauan kehadiran personel tim (Active Team) dan status sesi operasional secara real-time.
- **Automated Triage**: Alur kerja penilaian alert yang terstruktur untuk penanganan insiden yang lebih cepat.
- **Operations Reports**: Pembuatan laporan operasional dan audit sistem dengan format profesional (PDF/CSV).
- **Multi-Language Support**: Dukungan penuh untuk Bahasa Indonesia dan Bahasa Inggris pada seluruh antarmuka.
- **Demo Environment**: Mode simulasi untuk demonstrasi operasional tanpa konfigurasi infrastruktur yang rumit.

## 🛠️ Fungsi Aplikasi

Aplikasi ini berfungsi sebagai pusat kendali operasional (Command Center) bagi tim SOC untuk:
1. Mengagregasi data alert dari berbagai sumber telemetry.
2. Memantau kesehatan sistem dan distribusi beban kerja tim.
3. Mendokumentasikan langkah-langkah audit dan kepatuhan operasional.
4. Menjaga kesinambungan komunikasi antara analis selama pergantian shift.

## 🔄 Alur Kerja (Workflow)

1. **Ingestion**: Sistem menerima telemetry atau alert simulasi.
2. **Monitoring**: Analis memantau pergerakan alert melalui Dashboard Utama.
3. **Triage**: Alert dianalisis dan dikategorikan berdasarkan tingkat risiko dan urgensinya.
4. **Action**: Pengambilan tindakan respons atau eskalasi berdasarkan prosedur yang ditetapkan.
5. **Auditing**: Seluruh aktivitas dicatat dalam sistem audit untuk transparansi dan tinjauan pasca-insiden.

## 📖 Cara Penggunaan

1. **Akses Dashboard**: Masuk ke aplikasi melalui browser menggunakan URL yang tersedia.
2. **Navigasi Modul**: Gunakan sidebar untuk berpindah antara Dashboard, Alert Center, Triage, dan Operational Management.
3. **Manajemen Alert**: Pilih item dari Alert Grid untuk melihat detail forensik dan melakukan aksi triage.
4. **Monitoring Tim**: Periksa section 'Active Team' untuk melihat personel yang sedang bertugas.
5. **Ekspor Laporan**: Gunakan fitur ekspor pada modul Reports untuk menghasilkan dokumen ringkasan kegiatan.
